## Reporting a Vulnerability

We take the security of TubeTrend seriously. If you discover a security vulnerability, please report it responsibly.

### How to Report

**Please use GitHub's private vulnerability reporting feature** to report security issues:

1. Go to the [Security tab](https://github.com/fo0/tubetrend/security) of this repository
2. Click "Report a vulnerability"
3. Fill in the details of the vulnerability

Alternatively, you can report vulnerabilities through GitHub's private vulnerability reporting which is enabled for this repository.

### What to Expect

- **Acknowledgment**: We will acknowledge receipt of your report within 48 hours.
- **Updates**: We will provide updates on the progress of fixing the vulnerability.
- **Disclosure**: We will coordinate with you on the disclosure timeline.

### What We Ask

- Please do not open public issues for security vulnerabilities.
- Provide sufficient detail so we can reproduce and fix the issue.
- Give us reasonable time to address the vulnerability before public disclosure.

## Security Measures

This repository has the following security features enabled:

- **Dependabot Alerts & Security Updates**: Automated vulnerability detection and patching for dependencies
- **Secret Scanning & Push Protection**: Prevention of accidental secret/credential commits
- **CodeQL Analysis**: Automated code scanning for security vulnerabilities
- **Branch Protection**: Required pull requests for the main branch

Thank you for helping keep TubeTrend and its users safe!
